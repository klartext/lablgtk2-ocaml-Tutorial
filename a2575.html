GDK Event Types

The following data types are passed into event handlers by GTK+. For each data type listed, the signals that use this data type are listed. (See GObj.event_signals)

  • Gdk.Tags.event_type Gdk.event

    • any event

  • [`DELETE] Gdk.event

    • delete event

  • [`DESTROY] Gdk.event

    • destroy event

  • [`MAP] Gdk.event

    • map event

  • [`UNMAP] Gdk.event

    • unmap event

  • GdkEvent.Expose.t

    • expose event

  • GdkEvent.Motion.t

    • motion_notify event

  • GdkEvent.Button.t

    • button_press event

    • button_release event

  • GdkEvent.Key.t

    • key_press event

    • key_release event

  • GdkEvent.Crossing.t

    • enter_notify event

    • leave_notify event

  • GdkEvent.Focus.t

    • focus_in event

    • focus_out event

  • GdkEvent.Configure.t

    • configure event

  • GdkEvent.Property.t

    • property_notify event

  • GdkEvent.Selection.t

    • selection_clear event

    • selection_request event

    • selection_notify event

  • GdkEvent.Proximity.t

    • proximity_in event

    • proximity_out event
