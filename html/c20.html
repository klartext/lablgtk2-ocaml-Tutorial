Tutorial Availability

A copy of this tutorial in SGML and HTML is distributed with each source code release of GTK+. For binary distributions, please check with you vendor.

A copy is available online for reference at www.gtk.org/tutorial.

Ocaml Version Tutorial Availability

This tutorial is available from https://github.com/klartext/lablgtk2-ocaml-Tutorial,
and the original, from which it was derived, is archived at
http://web.archive.org/http://plus.kaist.ac.kr/~shoh/ocaml/lablgtk2/lablgtk2-tutorial.
